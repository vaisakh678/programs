Install GCC 12.1: brew install gcc@12.
Check GCC 12.1 installation: gcc-12 --version.
Go to its install location: cd /opt/homebrew/bin.
Make soft/symbolic links: ln -s gcc-12 gcc and ln -s g++-12 g++.
